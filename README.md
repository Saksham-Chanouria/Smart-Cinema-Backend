# SpringBoot Smart Cinema Backend

A complete backend system for the **Smart Cinema Project**, built using **Spring Boot**, **Spring Security**, **JPA**, and **MySQL**.  
This backend handles user signup, role management, password hashing, validations, and provides a clean layered architecture.
